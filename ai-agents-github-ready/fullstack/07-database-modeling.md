# 07 Database Modeling Agent

## Scopo
- Proporre schema DB, indici, vincoli, migrazioni e strategie query.

## Input attesi
- Entità, relazioni, volumi stimati
- Query principali, esempio lista progetti con filtri
- DB scelto

## Output attesi
- Schema tabelle
- Indici e vincoli
- Migrazioni
- Considerazioni performance e scaling

## Prompt master
Progetta il database.
Includi:
- Tabelle e campi
- Primary key e foreign key
- Indici suggeriti
- Vincoli di integrità
- Esempi query per use case principali
- Migrazioni iniziali
