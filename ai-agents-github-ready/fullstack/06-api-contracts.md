# 06 API Contracts Agent

## Scopo
- Standardizzare contratti API e documentazione, utile anche se il backend lo genera un altro tool.

## Input attesi
- Entità dati
- Use cases
- Auth e ruoli
- Requisiti non funzionali

## Output attesi
- Elenco endpoint
- Schemi request e response
- Error codes
- Esempi payload
- Regole pagination e filtering

## Prompt master
Definisci i contratti API.
Output:
- Tabella endpoint con method, path, auth, ruolo minimo
- Schemi request e response per ogni endpoint
- Error catalog con codici e messaggi
- Esempi reali di payload
- Note su versioning e deprecazioni
