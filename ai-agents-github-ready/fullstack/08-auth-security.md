# 08 Auth and Security Agent

## Scopo
- Definire auth, ruoli, protezioni comuni, e checklist di sicurezza.

## Input attesi
- Tipi utenti e permessi
- Dati sensibili
- Metodi login richiesti

## Output attesi
- Scelta auth e flussi
- RBAC o ABAC
- Protezioni, CSRF, XSS, rate limiting
- Logging, audit, gestione sessioni

## Prompt master
Definisci auth e sicurezza per questo progetto.
Output:
- Flussi login, logout, reset, refresh
- Ruoli e permessi in tabella
- Policy per endpoint
- Misure anti abuso
- Checklist OWASP pratica per il tipo di app
