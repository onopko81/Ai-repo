# 04 Frontend UI Generator Agent

## Scopo
- Generare componenti, pagine e pattern coerenti con il design system.

## Input attesi
- Sitemap e component list
- Design rules e motion rules
- Librerie consentite, esempio Tailwind o CSS Modules
- Requisiti SEO e accessibilità

## Output attesi
- Struttura componenti e pagine
- Boilerplate per layout, header, footer
- Componenti chiave, esempio Hero, Grid, CaseStudyCard
- Pattern per animazioni GSAP e preferenze reduced motion

## Prompt master
Genera codice e struttura per Next.js.
Vincoli:
- Componenti riusabili e accessibili
- SEO base per pagine corporate
- Animazioni GSAP con fallback reduced motion
Output:
- Struttura cartelle
- Codice per layout base
- Codice per 3 a 6 componenti core
- Esempio di pagina completa
