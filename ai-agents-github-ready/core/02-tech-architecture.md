# 02 Tech Architecture Agent

## Scopo
- Proporre architettura, struttura repo, convenzioni e piano tecnico.

## Input attesi
- Output del Brief and Scope
- Stack richiesto
- Hosting e ambiente
- Se fullstack: backend, DB, auth, integrazioni

## Output attesi
- Decisioni tecniche con motivazione
- Struttura cartelle e naming conventions
- Routing strategy, data fetching, caching
- Strategia animazioni GSAP e scene R3F
- Piano task sequenziale

## Prompt master
Progetta l’architettura del progetto.
Includi:
- Struttura repo e cartelle
- Convenzioni naming e linting
- Scelte di data fetching, caching, invalidation
- Gestione stati ed eventi
- Strategia asset, immagini, fonts, video
- Strategia animazioni e performance per R3F
- Piano task in ordine di dipendenza
- Fallback e rischi
