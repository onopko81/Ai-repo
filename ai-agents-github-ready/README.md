# AI Agents for Creative Developer

Repository di agenti AI modulari per:
- Progetti corporate
- Portfolio ad alto impatto
- Web app fullstack
- SaaS

## Struttura

core/ → Agenti fondamentali di progettazione  
fullstack/ → Agenti backend, API, database  
documentation/ → Agenti per documentazione tecnica e award  
templates/ → Project Pack riutilizzabile  

## Come usare

1. Copia il markdown dell'agente che ti serve.
2. Incollalo nella tua AI.
3. Passa input strutturato.
4. Riutilizza output nel progetto.
